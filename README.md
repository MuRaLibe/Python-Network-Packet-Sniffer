# Python Network Packet Sniffer

A professional GUI-based network packet sniffer built using Python, Scapy, and Tkinter.

## Features

- Live packet capture
- TCP/UDP/ICMP analysis
- DNS website detection
- Real-time packet statistics
- GUI dashboard
- Packet logging
- PCAP export support

## Technologies Used

- Python
- Scapy
- Tkinter
- Colorama
- Npcap

## Installation

pip install -r requirements.txt

## Run

python gui_sniffer.py

## Educational Purpose

This project is intended for ethical cybersecurity learning and educational use only.
